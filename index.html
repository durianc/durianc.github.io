Projects

Project 1: Neural Graph Collaborative Filtering-based Parking Space Recommendation System

  • Description: This project focused on developing a parking space recommendation system that utilizes neural graph collaborative filtering technology. By integrating user data and parking behavior patterns into a graph database, the system provides personalized parking space suggestions, enhancing user convenience and optimizing parking space usage.
  • Technologies Used: PyTorch, Neo4j, Pandas, Flask+Vue
  • Contribution: As the project leader, I oversaw the development process from initial design to final implementation. My responsibilities included guiding the architecture of the system, managing the development team, and ensuring that the recommendation algorithm met the project’s objectives and performance criteria.

Project 2: Video Identity Recognition

  • Description: Engineered an advanced identity recognition system that leverages real-time video streams, utilizing the YOLOv7 model for precise human detection and subsequent image segmentation. The system combines state-of-the-art Face Recognition and Person Re-ID algorithms to achieve a remarkable identification accuracy of 98%.
  • Core Idea: Integration of Face Recognition and Person Re-identification
  • Contribution: Key innovations include the implementation of standing detection and specifically delineated check-in zones to mitigate interference from unauthorized entities and minimize erroneous operations. Plans are underway to incorporate this system into the “Multimodal Big Data-Driven Platform for Research Efficiency Analysis”, aiming to facilitate a non-contact, camera-based authentication module in laboratory settings.

Project 3: Traffic Scene Object Detection Based on MMDetection

  • Description: The core objective of the project is to accurately and comprehensively detect all specified objects in traffic scene images by making adjustments in areas such as model selection, data augmentation, and loss function optimization.
